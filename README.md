# Homebrew Tap

Custom Homebrew tap for 0xtlt projects.

## Usage

```bash
brew tap 0xtlt/tap
brew install vitrail
brew install foldbar
```

## Available Casks

| Cask | Description |
|------|-------------|
| `foldbar` | Tiny macOS menu bar utility to fold away menu bar icons |
| `vitrail` | Lightweight macOS window layout manager |
